# Night Nuo Bundled Case Pack

This folder contains the full bundled example package for the Night Nuo storyboard workflow.

Everything in this folder is self-contained for sharing:

- the base script
- the stage-by-stage outputs
- the front-half and back-half shot-by-shot boards
- the full rough-board package
- the clean-pass rules

## What This Pack Demonstrates

1. How to protect the true climax from the middle action
2. How to keep an object-stakes line visible across the whole piece
3. How to convert public danger into private temptation
4. How to use stillness as a rank-change beat
5. How to let the final verdict line outrank the flashy inserts
