## 二、最终分镜主稿

## Scene 1：城市夜声

### 1-1

- 景别：`大全景`
- 画面 / 构图：城市夜景横切，高楼霓虹、地铁穿行、外卖员、夜店、急诊室同时在动，画面信息密但秩序清楚。
- 动作 / 表演：整座城市是忙的、活的、正常的。
- 机位 / 运动：横向流动或稳定横移，别做夸张运动。
- 声音 / 台词：城市低噪里混进极远的鼓点。
- cut / hold / 连续性：先把常态立住，再顺着城市流动把重心送向老城区。

### 1-2

- 景别：`中远景`
- 画面 / 构图：画面重心往老城区边缘收，霓虹和现代感减弱，空间开始变旧。
- 动作 / 表演：不是角色在演，是空间气质在变。
- 机位 / 运动：继续顺势收缩，不要突然跳轴。
- 声音 / 台词：鼓点更清楚一点，但还没有源头。
- cut / hold / 连续性：把观众从“公共都市”平稳送向“异常边缘”。

### 1-3

- 景别：`中景`
- 画面 / 构图：老城区偏暗角落或巷口边缘，留出明显阴影和空处。
- 动作 / 表演：空间静下来，像在等什么。
- 机位 / 运动：尽量稳，靠构图而不是运镜制造不安。
- 声音 / 台词：鼓声更近一点。
- cut / hold / 连续性：让声音先接管空间，再把规则砸下来。

### 1-4

- 景别：`中景 held`
- 画面 / 构图：空街口或暗巷口留白，画面本身不要塞满信息。
- 动作 / 表演：画面尽量不动，让观众听规则。
- 机位 / 运动：静止。
- 声音 / 台词：旁白：“听见鼓声，别回头。”
- cut / hold / 连续性：要留住这句规则，不能急着切怪象。
