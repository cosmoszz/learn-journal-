## Scene 7：面匣失守，禁面翻出

时间：

- [yenuo_script_utf8.txt:91](C:/Users/Administrator/Documents/Playground/yenuo_script_utf8.txt#L91) 到 [yenuo_script_utf8.txt:104](C:/Users/Administrator/Documents/Playground/yenuo_script_utf8.txt#L104)

### 7-1

- 景别 / 画面：`中远景`，战圈中段。女主转身回护面匣，纸脸女孩从侧后撞入，前辈被另一股黑影牵住。
- 动作 / 表演：三股动作同时汇向面匣，但画面主焦点始终是“护匣链条断开”。
- 声音 / 台词：打斗环境声继续，不急着上台词。
- 切换职责：先把失守的因果讲清，不能直接跳匣子落地。

### 7-2

- 景别 / 画面：`中景`，几只黑影同时扑上面匣，锁扣在画面中央崩开。
- 动作 / 表演：扑击是挤压感，不是炫动作；锁扣断裂是这镜真正命中点。
- 声音 / 台词：锁扣崩裂声必须脆。
- 切换职责：把 stakes 从“快要失守”推进到“已经失守”。

### 7-3

- 景别 / 画面：`插入中近景`，黑木匣脱手，翻进雨水，水花打起。
- 动作 / 表演：匣子翻滚方向要明确，朝男主所处区域去。
- 声音 / 台词：沉重落水声。
- 切换职责：把动作因果顺进下一镜的禁面翻出。

### 7-4

- 景别 / 画面：`特写插入`，残破古傩面在积水里缓缓翻过来，停在男主鞋尖前。
- 动作 / 表演：翻面动作不要急，必须让观众“看见它出现”。
- 声音 / 台词：战场噪声像被抽远半层。
- 切换职责：这是整场命门，要给 hold。

### 7-5

- 景别 / 画面：`中近景`，女主脸色骤变，朝男主厉喝。
- 动作 / 表演：不是普通提醒，而是规矩被碰到的失态。
- 声音 / 台词：女主：“别碰！”
- 切换职责：第一道禁令落下，把面具从“道具”变成“禁物”。

### 7-6

- 景别 / 画面：`中近景 / 反应镜头`，前辈也失声看向男主与脚边面具。
- 动作 / 表演：他的失态要比女主更重，因为他知道后果。
- 声音 / 台词：前辈：“那张面不能戴！”
- 切换职责：第二道禁令加码，为下一场的越线建立重量。

## Scene 8：代伤、诱惑、扣面

时间：

- [yenuo_script_utf8.txt:106](C:/Users/Administrator/Documents/Playground/yenuo_script_utf8.txt#L106) 到 [yenuo_script_utf8.txt:117](C:/Users/Administrator/Documents/Playground/yenuo_script_utf8.txt#L117)

### 8-1

- 景别 / 画面：`中景`，黑影再次扑下，方向直指男主所在位置。
- 动作 / 表演：动作线要单纯，让观众清楚“危险现在就是落向他”。
- 声音 / 台词：黑影扑落的风切声。
- 切换职责：把禁令后的危机重新压回来。

### 8-2

- 景别 / 画面：`中景`，女主横切进男主与黑影之间，整个人像一刀插进画面。
- 动作 / 表演：她不是摆 pose，是纯本能护住。
- 声音 / 台词：冲撞、撕裂几乎同时到。
- 切换职责：建立“她替他挨”。

### 8-3

- 景别 / 画面：`中近景`，女主肩头被撕开，半跪；男主僵住，脚边面具仍在。
- 动作 / 表演：这镜里最好同时看到伤、男主、面具三者关系。
- 声音 / 台词：女主压住痛感；男主呼吸发颤。
- 切换职责：把选择结构一次钉死。

### 8-4

- 景别 / 画面：`中近景 held`，男主低头看面，背景的风雨声忽然被抽掉半秒。
- 动作 / 表演：不是发狠，是被逼进死角。
- 声音 / 台词：风声静；低语进耳：“你不戴，她会死。”
- 切换职责：从外部打斗转成私人诱惑，这镜必须留。

### 8-5

- 景别 / 画面：`近景`，男主手猛地抓住面具。
- 动作 / 表演：动作要短、突、怕自己后悔。
- 声音 / 台词：低语接一句：“我替你杀。”
- 切换职责：把心理选择落成身体动作。

### 8-6

- 景别 / 画面：`近景到结果宽镜`，男主将面具扣上，下一瞬整条街灯全灭。
- 动作 / 表演：扣面的动作不拖，结果要重。
- 声音 / 台词：灯灭声、世界抽空。
- 切换职责：个人越线直接改写世界。

## 四、这一版真正能拿去干什么

2. 可以拿去做 scene 7-10 的粗 animatic

1. Scene 7 的禁面翻面必须留
2. Scene 8 的风静低语必须留
3. Scene 10 的“那不是神”必须站住
